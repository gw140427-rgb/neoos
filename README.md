# 📱 Note20 Windows 7 Optimizer
안드로이드 Termux 환경에서 Windows 7을 가장 빠르고 안정적으로 돌리기 위한 프로젝트입니다.

### ✨ 설정 포인트
- **Memory:** 2GB (8GB 램 중 시스템 점유율 고려 최적화)
- **CPU:** `max` 플래그 사용으로 스냅드래곤 865 성능 극대화
- **Network:** `e1000` 드라이버 적용 (인터넷 연결 호환성 높음)

### ⚠️ 주의사항 (필독!)
- 윈도우 7 내에서 **TLS 1.2**를 수동으로 활성화해야 인터넷 페이지가 열립니다.
- 이미지 파일 이름은 `win7.qcow2`로 맞춰주세요.
